# 📊 Telecom X - Predicción de Churn con Programación Orientada a Objetos

Este proyecto tiene como objetivo **analizar y predecir la evasión de clientes (Churn)** en la empresa ficticia **Telecom X**, utilizando técnicas de **análisis de datos y Machine Learning** implementadas con **Python y Programación Orientada a Objetos (POO)**.

El notebook desarrolla un flujo completo de ciencia de datos que incluye:

- 📥 Carga y preparación de datos  
- 🧹 Limpieza y transformación  
- 📊 Análisis exploratorio  
- 🤖 Entrenamiento de modelos de Machine Learning  
- 📈 Evaluación del rendimiento de los modelos  

El objetivo es **identificar clientes con alta probabilidad de abandonar el servicio** para que la empresa pueda implementar **estrategias de retención más efectivas**.

---

# 🎯 Objetivo del Proyecto

El proyecto busca desarrollar un modelo capaz de **predecir el abandono de clientes (churn)** utilizando información histórica de clientes de Telecom X.

Este análisis permite:

- Identificar **clientes en riesgo de abandonar el servicio**
- Comprender **qué variables influyen en el churn**
- Apoyar la **toma de decisiones estratégicas del negocio**

---

# 🗂️ Dataset

El proyecto utiliza el dataset de clientes de Telecom X, el cual incluye información sobre:

- ID del cliente  
- Género  
- Adulto mayor  
- Tipo de contrato  
- Servicios contratados  
- Cargos mensuales  
- Cargos totales  
- Antigüedad del cliente  
- Método de pago  
- Estado de abandono (**Churn**)

Este conjunto de datos permite analizar el comportamiento de los clientes y entrenar modelos de predicción.

---

# ⚙️ Metodología del Proyecto

El proyecto sigue un flujo típico de **Machine Learning aplicado a análisis de churn**.

---

# 🧹 1. Preparación de los Datos

En esta etapa se realizó:

- Limpieza de datos
- Tratamiento de valores faltantes
- Conversión de variables categóricas
- Normalización de variables numéricas

También se preparó el dataset para su uso en modelos de Machine Learning.

---

# 📊 2. Análisis Exploratorio de Datos

Se analizaron las variables más importantes para entender el comportamiento de los clientes.

Entre los análisis realizados se encuentran:

- Distribución de churn
- Análisis por tipo de contrato
- Relación entre cargos mensuales y abandono
- Análisis de antigüedad del cliente

Este análisis permitió identificar **patrones importantes en el abandono de clientes**.

---

# 🧠 3. Modelos de Machine Learning

Se implementaron modelos de clasificación para predecir la probabilidad de churn.

El flujo incluyó:

1. Separación de datos en **entrenamiento y prueba**
2. Entrenamiento de modelos de clasificación
3. Evaluación del rendimiento del modelo

Los modelos permiten estimar la probabilidad de que un cliente abandone el servicio.

---

# 📈 Evaluación del Modelo

Los modelos fueron evaluados utilizando métricas de Machine Learning como:

- **Accuracy**
- **Precision**
- **Recall**
- **F1 Score**
- **Matriz de confusión**

Estas métricas permiten medir la capacidad del modelo para **identificar correctamente a los clientes que podrían abandonar el servicio**.
